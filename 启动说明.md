# 🎵 AI DJ 项目启动指南

## 🚀 一键启动（推荐）

双击运行项目根目录下的 `start.bat` 文件，会自动启动：
1. 后端服务器（端口 8080）
2. 前端开发服务器（端口 3030）

然后在浏览器访问：**http://localhost:3030**

---

## 🔧 手动启动

### 1. 启动后端服务器
```bash
npm start
```

### 2. 启动前端开发服务器
```bash
cd frontend
npm run dev
```

### 3. 访问应用
打开浏览器访问：**http://localhost:3030**

## 📝 注意事项

1. **首次启动**：确保已安装依赖
   ```bash
   npm install
   cd frontend && npm install
   ```

2. **环境变量**：确保根目录有 `.env` 文件，包含必要的API密钥

3. **端口占用**：如果端口被占用，请先关闭占用端口的程序

## 🐛 故障排除

### 问题：页面显示 "Cannot GET /"
**解决方案**：
- 确保后端服务器（8080端口）正在运行
- 使用 `start.bat` 一键启动

### 问题：修改代码后没有自动刷新
**解决方案**：
- 已配置 `usePolling: true`，应该会自动刷新
- 如果还是不行，手动刷新：Ctrl+Shift+R

### 问题：音频可视化不工作
**解决方案**：
- 确保浏览器支持Web Audio API（Chrome、Edge、Firefox都支持）
- 检查浏览器控制台是否有错误信息
- 确保音乐正在播放
