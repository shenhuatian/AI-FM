# 💻 在电脑上使用 PWA - 完整指南

## 🎯 您的需求
- ✅ 只在电脑上使用
- ✅ 不需要手机
- ✅ 像桌面软件一样使用

## 📝 简单理解

**PWA 在电脑上 = 一个独立的桌面应用**

就像您安装了 QQ、微信、网易云音乐，双击图标就能打开，有自己的窗口。

**您的 Claudio FM 也可以这样！**

---

## 🚀 在电脑上安装 Claudio FM PWA

### 步骤 1：启动服务
```bash
# 双击运行
start-all.bat
```

### 步骤 2：打开浏览器
打开 **Chrome** 或 **Edge** 浏览器，访问：
```
http://localhost:8080
```

### 步骤 3：安装 PWA
您会看到地址栏右侧有一个 **安装图标** ⊕

点击后会弹出提示：
```
安装 Claudio FM？
[取消]  [安装]
```

点击 **[安装]**

### 步骤 4：完成！
安装后：
- ✅ 桌面上会出现 **Claudio FM** 图标
- ✅ 开始菜单也会有
- ✅ 双击图标就能打开
- ✅ 像独立软件一样运行

---

## 🎨 安装后的效果

### 打开方式
1. **桌面图标** - 双击打开
2. **开始菜单** - 搜索 "Claudio FM"
3. **任务栏** - 可以固定到任务栏

### 窗口样式
- 独立窗口
- 没有浏览器地址栏
- 没有书签栏
- 就像一个真正的桌面软件

---

## 🔧 如果找不到安装按钮

### 方法 1：手动安装（Chrome）
1. 点击浏览器右上角 **⋮** (三个点)
2. 找到 **"安装 Claudio FM..."**
3. 点击安装

### 方法 2：手动安装（Edge）
1. 点击浏览器右上角 **⋯** (三个点)
2. 找到 **"应用"** → **"将此站点作为应用安装"**
3. 点击安装

---

## 📊 PWA vs 浏览器标签页

| 特性 | 浏览器标签页 | PWA 独立窗口 |
|------|-------------|-------------|
| 地址栏 | 有 | 没有 |
| 书签栏 | 有 | 没有 |
| 独立图标 | 没有 | 有 |
| 桌面快捷方式 | 没有 | 有 |
| 体验 | 网页 | 桌面软件 |
